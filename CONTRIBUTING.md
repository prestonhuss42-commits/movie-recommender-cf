# Contributing

Thanks for your interest in contributing.

## Development setup
1. Create a Python environment and install dependencies.
2. Train the model to generate `artifacts/cf_model.joblib`.
3. Run API and frontend locally.

## Pull request checklist
- Keep changes focused and minimal.
- Run local sanity checks for endpoints.
- Update README/docs if behavior changes.
- Do not commit secrets or `.env` files.

## Style
- Follow existing Python and React patterns.
- Prefer clear naming and small functions.
- Keep ML/business logic in modular services.
